# HR Assistant Agent

An AI-powered HR assistant that answers employee queries about company policies, leaves, and benefits using Google Gemini AI.

## Features
- AI-powered HR responses using Google Gemini
- Local fallback knowledge base
- Interactive chat interface
- Quick question buttons
- Professional HR responses

## Setup
1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Add Google API key to `.env` file
4. Run: `streamlit run app.py`

## Tech Stack
- Streamlit
- Google Gemini AI
- Python
